# Walnut Creek (Treat Blvd) – Modern Preschool Website (Static)

This is a fast, one-page static website you can deploy using GitHub Pages.

## Run locally
Just open `index.html` in your browser.

## Deploy with GitHub Pages
1. Create a new GitHub repo
2. Upload these files (`index.html`, `styles.css`, `script.js`, `README.md`)
3. Go to **Settings → Pages**
4. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main** / **root**
5. Save — your site will publish to a GitHub Pages URL.

## Customize
- Update title, copy, and the contact details in `index.html`
- In `script.js`, set `DESTINATION_EMAIL` to your real inbox
- Add images if you want (hero background, classroom photos, etc.)
